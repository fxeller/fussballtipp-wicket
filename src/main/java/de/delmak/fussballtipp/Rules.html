Spielregeln

Tipps auf Spiele

  • Die Spieltipps sind über den Menüpunkt 'Tipps' erreichbar.
  • Tipps für ein Spiel können bis 30 Minuten vor Spielbeginn abgegeben oder geändert werden.
  • Punkte für das Spiel gibt es, wenn das Ergebnis eingetragen wurde.
  • Für die richtige Tendenz (Sieg, Unentschieden, Niederlage) gibts einen Punkt.
  • Wer die Tendenz richtig hat, kriegt nochmals jeweils einen Punkt, wenn er die Tore der Heim- oder Gastmannschaft richtig hat.
  • Das Maximum an Punkten pro Spiel beläuft sich daher auf 3 Punkte.
  • Als Endergebnis zählt der Spielstand inklusive Tore aus einem eventuellen Elfmeterschießen.

    • Tipps auf Gruppensieger und Weltmeister

  • Die Gruppensiegertipps sind über den Menüpunkt 'Tipps' und dann über 'Sondertipps' erreichbar.
  • Tipps für einen Gruppensieger können bis 30 Minuten vor dem ersten Spiel der jeweiligen Gruppe abgegeben oder geändert werden.
  • Der Weltmeistertipp kann bis 30 Minuten vor Spielbeginn des ersten Achtelfinales abgegeben oder geändert werden.
  • Punkte für Gruppensieger und Weltmeister gibt es, wenn die jeweiligen Sieger eingetragen wurden.
  • Für einen richtigen Gruppensieger gibt es 3 Punkte, für den richtigen Weltmeister 5.
